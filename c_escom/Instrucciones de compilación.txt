Compilación y ejecución de programas en C
=========================================

Este documento explica cómo compilar y ejecutar un archivo en lenguaje C utilizando la terminal y el editor Visual Studio Code (VSCode). También se incluye un enlace para configurar el compilador GNU (GCC) y el entorno de desarrollo.

1. Compilar y ejecutar desde la terminal
----------------------------------------

Para compilar un archivo llamado programa.c desde la terminal:

1. Abre la terminal.
2. Navega al directorio donde se encuentra el archivo:
   cd /ruta/del/archivo
3. Compila el archivo usando gcc:
   gcc programa.c -o programa
4. Ejecuta el programa compilado:
   ./programa

2. Compilar y ejecutar desde Visual Studio Code
-----------------------------------------------

Para compilar y ejecutar programas en C desde VSCode:

1. Instala la extensión "C/C++" de Microsoft.
2. Instala el compilador GCC (por ejemplo, MinGW-w64 en Windows o build-essential en Linux).
3. Crea un archivo programa.c y escribe tu código.
4. Abre la terminal integrada en VSCode (Ctrl + `).
5. Compila y ejecuta como lo harías en la terminal:
   gcc programa.c -o programa
   ./programa

3. Configuración del entorno
----------------------------

Para configurar correctamente el compilador GNU y VSCode, consulta el siguiente artículo:

Guía paso a paso:
https://programmerclick.com/article/9587931241/

Este recurso explica cómo instalar MinGW-w64, configurar variables de entorno, y preparar VSCode para compilar y depurar programas en C.

Conclusión
----------

Compilar y ejecutar programas en C es sencillo tanto desde la terminal como desde VSCode, siempre que el entorno esté correctamente configurado. Usar herramientas modernas como VSCode puede facilitar la escritura, depuración y organización del código.
